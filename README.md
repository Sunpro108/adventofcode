
## Day02

### Day021

**核心问题：** 

统计字符串中字符出现的次数

将包含有两个相同字母的字符串数（numTwo）与包含3个相同字母的字符串数(numThree)相乘，作为**checksum**

**思路：**

- 统计字符串中各字母的出现次数

- 确认是否有字母正好出现两次，有的话，numTwo加1
- 确认是否有字母正好出现三次，有的话，numThree加1

- 遍历所有字符串，得到最终的numTwo,numThree，二者相乘
